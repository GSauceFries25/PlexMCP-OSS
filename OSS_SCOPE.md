# PlexMCP OSS Scope

This repository contains the source-available, self-hostable core of PlexMCP.

## Included

- MCP server orchestration
- Registry and routing
- Permissions, roles, and audit logs
- Local and self-hosted deployments

## Not Included

- PlexMCP Cloud operational tooling
- Billing activation
- Internal admin dashboards
- Managed service infrastructure

PlexMCP Cloud is built on top of this core and includes additional proprietary services.
