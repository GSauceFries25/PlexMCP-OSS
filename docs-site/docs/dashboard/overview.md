# Dashboard Overview

The PlexMCP dashboard is your central hub for managing MCPs, API keys, team members, and monitoring usage.

## Accessing the Dashboard

Visit [dashboard.plexmcp.com](https://dashboard.plexmcp.com) and sign in with your account.

## Home Screen

When you log in, you'll see your organization overview:

### Usage Summary
- **Requests This Period**: Total API calls made
- **Active MCPs**: Number of connected MCP servers
- **API Keys**: Active keys count
- **Team Members**: Number of users in your organization

### Quick Actions
- **Add MCP**: Register a new MCP server
- **Create API Key**: Generate a new API key
- **Invite Member**: Add a team member

### Recent Activity
A chronological list of recent events:
- API calls made
- Keys created or revoked
- Team members added
- MCPs added or modified

### Usage Chart
Visual representation of your API usage over time, with options to view:
- Last 24 hours
- Last 7 days
- Last 30 days
- Custom date range

## Navigation

The sidebar provides access to all sections:

| Section | Purpose |
|---------|---------|
| **Dashboard** | Overview and quick stats |
| **MCPs** | Manage MCP servers |
| **API Keys** | Create and manage keys |
| **Team** | Manage team members |
| **Billing** | Subscription and invoices |
| **Settings** | Organization configuration |

## Organization Switcher

If you belong to multiple organizations:

1. Click your organization name in the sidebar
2. Select a different organization
3. The dashboard will refresh with that org's data

## Search

Use the search bar (or press `/`) to quickly find:
- MCPs by name
- API keys
- Team members
- Documentation pages

## Notifications

The bell icon shows recent notifications:
- Usage warnings (approaching limits)
- MCP health alerts
- Team activity
- Billing reminders

## Profile Menu

Click your avatar to access:
- **Profile Settings**: Update your name and avatar
- **Account Settings**: Email and password
- **Sign Out**: Log out of your account

## Mobile Access

The dashboard is fully responsive:
- Navigation collapses to a hamburger menu
- Tables become scrollable cards
- Touch-friendly interface throughout

## Keyboard Shortcuts

Power users can navigate quickly:

| Shortcut | Action |
|----------|--------|
| `G D` | Go to Dashboard |
| `G M` | Go to MCPs |
| `G K` | Go to API Keys |
| `G T` | Go to Team |
| `G B` | Go to Billing |
| `G S` | Go to Settings |
| `/` | Focus search |
| `?` | Show shortcuts help |
| `Esc` | Close modal/menu |
